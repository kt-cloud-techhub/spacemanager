# SpaceManager AI-DLC Workshop

이 워크스페이스는 **SpaceManager** 프로젝트의 기획, 설계, 구현 및 운영의 모든 과정을 **AI-DLC(Software Development Life Cycle)** 방법론에 따라 관리합니다.

## 🚀 빠른 시작

1. **문서 확인**: `aidlc-docs/` 폴더 내의 각 단계별 마일스톤 문서를 확인하세요.
2. **코드 위치**: 모든 서비스 소스 코드는 `apps/spacemanager/` 폴더에 위치합니다.
3. **워크플로우**: `antigravity.md` 파일은 이 워크스페이스의 모든 작업 원칙과 워크플로우를 정의합니다.

## 📂 주요 폴더 구조

- **`apps/`**: 애플리케이션 소스 코드 (Spring Boot Backend, React Frontend)
- **`aidlc-docs/`**: 프로젝트 모든 문서화 (PRD, 설계서, 로드맵 등)
- **`rules/`**: AI 코딩 어시스턴트를 위한 공통/커스텀 규칙
