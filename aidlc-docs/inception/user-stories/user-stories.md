# [User Stories] SpaceManager 사용자 스토리

본 문서는 사용자의 관점에서 **SpaceManager**가 제공해야 하는 핵심 가치를 시나리오 형태로 정의합니다.

---

## 1. 관리자 (Admin) - HR/총무 담당자

### 1.1 마스터 데이터 관리
*   **AS A** 관리자, **I WANT TO** 우리 회사의 복잡한 조직도(본부-담당-팀)를 일괄 업로드하고 싶다, **SO THAT** 일일이 입력하는 수고를 덜고 싶다.
*   **AS A** 관리자, **I WANT TO** 조직 트리에서 특정 조직을 추가하거나 위치를 변경하고 싶다, **SO THAT** 잦은 조직 개편에 유연하게 대응하고 싶다.

### 1.2 상면 및 구역 설정
*   **AS A** 관리자, **I WANT TO** 사무실 평면도 이미지를 업로드하고 그 위에 좌석 좌표를 설정하고 싶다, **SO THAT** 직원들이 실제 사무실과 동일한 지도를 보게 하고 싶다.
*   **AS A** 관리자, **I WANT TO** 층별로 각 팀이 사용할 구역(Boundary)을 다각형으로 그리고 싶다, **SO THAT** 팀별 가용 공간을 명확히 정의하고 싶다.

### 1.3 배치 시뮬레이션
*   **AS A** 관리자, **I WANT TO** 임원의 위치를 고정한 뒤 관련 팀들이 근처에 배치되도록 자동 제안을 받고 싶다, **SO THAT** 임원과 실무 팀 간의 물리적 거리를 최적화하고 싶다.
*   **AS A** 관리자, **I WANT TO** 현재 자리 배치 대비 이동이 가장 적은 안을 확인하고 싶다, **SO THAT** 이사 비용과 혼란을 최소화하고 싶다.

---

## 2. 사용자 (Member) - 임원 및 직원

### 2.1 본인 구역 확인
*   **AS A** 직원, **I WANT TO** 로그인 시 내 팀이 이번에 할당받은 구역이 어디인지 지도로 확인하고 싶다, **SO THAT** 내가 가서 앉을 수 있는 전체적인 영역을 파악하고 싶다.

### 2.2 좌석 선택 및 예약
*   **AS A** 직원, **I WANT TO** 우리 팀 구역 내의 빈 자리를 클릭하여 내 자리로 확정하고 싶다, **SO THAT** 내가 선호하는 위치(창가, 복도 등)를 직접 고르고 싶다.
*   **AS A** 임원, **I WANT TO** 내 팀 구역 내에서 임원석으로 지정된 전용 자리를 예약하고 싶다, **SO THAT** 업무 몰입도를 높일 수 있는 독립된 공간을 확보하고 싶다.

### 2.3 배치 결과 열람
*   **AS A** 직원, **I WANT TO** 전체 배치도를 통해 다른 팀이 어디에 있는지, 동료는 어디에 앉았는지 검색하고 싶다, **SO THAT** 사내 협업 지점을 쉽게 찾고 싶다.

---

## 3. 시스템 및 인프라 (NFR 포함)

*   **AS A** 시스템 사용자, **I WANT TO** 사내 SSO를 통해 별도 가입 없이 로그인하고 싶다, **SO THAT** 계정 관리의 번거로움을 줄이고 싶다.
*   **AS A** 관리자, **I WANT TO** 대규모 인원(수천 명)의 자리가 동시에 확정되어도 시스템이 멈추지 않기를 바란다, **SO THAT** 대규모 이사 시즌에 안정적으로 서비스를 운영하고 싶다.
